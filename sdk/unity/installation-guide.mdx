Installing the LMNT Unity Package follows the same pattern as other Unity packages. This guide
will walk you through the process.

## Add the LMNT Unity Package to your project
There are two ways to add the LMNT Unity Package to your Unity project: from the Unity Asset Store or
via direct download.

### Installing from the Unity Asset Store
- Navigate to the the [Unity Asset Store](https://assetstore.unity.com/) and search for "LMNT" in
the search box
- Select the LMNT package from the search result page
- Press the "Add to My Assets" button on the right side
- Accept the Asset Store Terms and License agreement
- Press the "Open in Unity" button in the popup banner
- Follow [these instructions](https://docs.unity3d.com/2022.3/Documentation/Manual/upm-ui-import.html) from the Unity documentation

### Installing via direct download
- Download the LMNT Package from [this URL](https://r.lmnt.com/unity/LMNT-Unity-package-0.1.0.tgz)
- Follow [these instructions](https://docs.unity3d.com/2022.3/Documentation/Manual/upm-ui-tarball.html) from the Unity documentation
